## Project Description
A comprehensive tool for generating astrological birth charts and transit data completely offline. It runs directly in your browser without any server-side dependencies.

## Features
- **Mandarin UI**: Fully localized interface for Chinese speakers.
- **Birth Chart Calculation**: Precise calculation of planetary positions and houses.
- **Transit Chart Support**: Real-time transit overlays and house detection.
- **Multiple House Systems**: Supports Placidus, Whole Sign, and Equal House systems.
- **Visual Analysis**: Interactive SVG chart with element/modality energy analysis.
- **Data Export**: Export charts as PNG images or session history as JSON.
- **Offline Functionality**: No internet connection required once files are downloaded.
